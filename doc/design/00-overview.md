# Pasta Atlas

Pasta Atlas is a web application for uploading and browsing Factorio map screenshots generated by the [mapshot](https://mods.factorio.com/mod/mapshot) MOD.

## Features

- Upload mapshot output (tile images + `mapshot.json`) directly from the browser
- Browse maps uploaded by any user on the top page
- View a map with pan, zoom, surface switching, generation switching, train station overlay, and tags overlay

## Tech Stack

| Layer | Technology |
|---|---|
| Backend | Ruby 4.0+, Hanami 2.3+, ROM.rb, Dry.rb |
| Frontend | Solid.js, Bulma, Leaflet.js |
| Database | PostgreSQL |
| Storage | AWS S3 + CloudFront |
| Infrastructure | Terraform |
