# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it through [GitHub Private Vulnerability Reporting](https://github.com/sakuro/pasta_atlas/security/advisories/new) rather than opening a public issue.

This allows us to review and address the issue before any public disclosure.

## What to Include

- A description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Any relevant logs, screenshots, or proof-of-concept code

## Response

We will acknowledge your report within a few business days and aim to provide a fix or mitigation as promptly as possible. We will keep you informed of our progress throughout the process.
