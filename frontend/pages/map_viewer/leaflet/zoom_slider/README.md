# ZoomSliderControl

A Leaflet control that replaces the default zoom buttons with a vertical slider.
Supports knob dragging, track clicking, and +/− buttons. The slider range
adjusts automatically when the map's min/max zoom changes.

This is a TypeScript rewrite based on the concept of
[Leaflet.zoomslider](https://github.com/kartena/Leaflet.zoomslider)
by Kartena AB and Mattias Bengtsson. The original plugin's license is preserved
in LICENSE.
