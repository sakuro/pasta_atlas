# BoxZoomControl

A Leaflet control that adds a box-zoom button to the map. Clicking the button
enters box-zoom mode; dragging on the map then zooms to the selected rectangle.
Press Escape to cancel.

This is a TypeScript rewrite based on the concept of
[L.Control.BoxZoom](https://github.com/gregallensworth/L.Control.BoxZoom)
by Greg Allensworth. The original plugin's license is preserved in LICENSE.
